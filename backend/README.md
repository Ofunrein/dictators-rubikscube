# Backend (Sprint 1)

Sprint 1 is frontend-first. This backend folder contains C++ placeholders for future cube-state, move, and solver implementation.

## API Scaffold (Sprint 2 prep)

An API contract + mock/skeleton server scaffold is available in [`backend/api`](./api):

- `openapi.yaml`: contract and examples
- `npm run mock`: spec-driven mock server (`:4010`)
- `npm run serve`: validation + route skeleton server (`:4011`)
